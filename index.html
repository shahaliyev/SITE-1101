🗂️ Course Outline (by Week)

  1. Course intro; Defining Engineering/Technology; System concepts
  2. Information concepts; What is an IS; IS components
  3. Hardware I: Binary/hex, conversions, ops; Boolean logic — HW1 out
  4. Hardware II: CPU, I/O, memory & storage — Team Project 1 out
  5. Software I: Systems vs Apps; OS; SE basics
  6. Software II: Algorithms, conditions/loops, recursion; first program — HW2 out (C++)
  7. Software III: Languages, paradigms, compiler vs interpreter
  8. Telecom & Networks; services & applications — Team Project 2 (Hour of Code) out
  9. Internet & Web I: OSI, Internet stack, IP, DNS
  10. Internet & Web II: HTTP, URL, markup; intranet/extranet; cloud — Individual Project 3 (Portfolio) out
  11. Data Mgmt: ER modeling; relational model; indexes; sorting/searching
  12. DBMS: SQL, NoSQL, transactions/ACID; big data & BI
  13. AI/ML & Generative AI — TBA
  14. SDLC, Agile, DevOps; feasibility & planning
  15. Ethics & social responsibility; privacy, crime, workplace issues
